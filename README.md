# Ola-Data-Analytics-Project
An end-to-end Power BI data analytics project based on Ola ride data. This project includes data cleaning, transformation, interactive dashboards and business insights to help understand ride performance, revenue, cancellations, and customer trends.

## Project Objective
To analyze Ola ride booking data using SQL and Power BI, identify key business trends and performance metrics, and develop interactive dashboards that provide actionable insights into bookings, revenue, cancellations, vehicle performance, and customer satisfaction for better business decision-making.
